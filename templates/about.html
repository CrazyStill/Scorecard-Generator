About Scorecard Creator

Overview

Scorecard Creator is a desktop application for generating print-ready scorecard PDFs for various sporting events. Upload your Word template and a CSV data file, configure the field mapping once, and generate a complete PDF batch whenever you need it. Everything runs locally on this PC — no internet connection required.

How It Works

  1. Upload a New Template:
    • Click Upload Template and enter a sport name and template name.
    • Upload your Word template (.docx) with placeholders for the data fields (e.g. TEAMNAME, DATEHERE).
      Tip: Use a consistent naming convention and avoid using _N (underscore + number) suffixes in your field names — those are reserved for multi-card positioning.
    • Upload a CSV file that defines the column headers for your data.
    • Optionally upload a static back design (PDF) to be merged onto the reverse of every scorecard page.
  2. Map CSV Headers to Placeholders:
    • After uploading, the mapping screen opens automatically.
    • For each CSV column header, enter the matching placeholder text from your Word template.
    • Set how many scorecards are printed per page (1–4). For multi-card pages, the program appends _1, _2, etc. to each placeholder automatically (e.g. TEAMNAME_1, TEAMNAME_2).
    • Click Save Mapping. This only needs to be done once per template.
  3. Generating Scorecards:
    • On the Generate screen, click Download CSV Template to get a blank CSV with the correct headers.
    • Fill in your match data in Excel or any spreadsheet program and save the file.
    • Upload the completed CSV and click Generate Scorecard PDF.
    • The program replaces all placeholders with your data, converts each page to PDF, and merges them into a single file.
    • When complete, the finished PDF is saved to your Downloads folder and you are returned to the home screen.
  4. File Storage:
    • Templates are stored in your Windows AppData folder (%APPDATA%\ScorecardCreator\SCTEMP\) and are preserved between updates.
    • Temporary files created during PDF generation are automatically cleaned up after each run.

Step-by-Step Guide

Step 1: Upload a New Template

On the Upload Template screen:

  • Enter the sport name (e.g. Soccer, Basketball) and a unique template name.
  • Select your Word document (.docx). Make sure it contains the correct placeholder text.
  • Select your CSV template file — this should have column headers matching your data fields.
  • If you have a static back design, select the PDF and choose Yes for the back design option.
  • Click Upload Template to save and proceed to mapping.

Step 2: Configure the Mapping

On the mapping screen:

  • For each CSV column header, type the exact placeholder text used in your Word template.
  • For multi-card pages, the program will automatically append _1, _2, etc. when filling in the template — do not add these suffixes yourself in the mapping.
  • Set the number of scorecards per page (typically 1 for basketball, 4 for soccer).
  • Click Save Mapping.

Step 3: Generate Scorecards

On the Generate screen:

  • Click Download CSV Template to get a blank file with the correct headers.
  • Open the file in Microsoft Excel, fill in your match data row by row, and save it.
  • Upload the completed CSV and click Generate Scorecard PDF.
  • The program will process all rows and save the finished PDF to your Downloads folder.

Step 4: Deleting a Template

  • From the home screen, click Delete on any template card to permanently remove it and all its associated files.

Technical Details

Scorecard Creator runs entirely on your PC and uses the following components:

  • Microsoft Word: Required for DOCX-to-PDF conversion via COM automation.
  • docx2pdf / pywin32: Drives the Word conversion process in the background.
  • PyPDF2: Merges the individual page PDFs into the final output file.
  • python-docx: Reads and fills in the Word template placeholders.

Frequently Asked Questions

Q: Do I need an internet connection?
A: No. Scorecard Creator runs entirely offline on this PC.

Q: Do I need Microsoft Word installed?
A: Yes. Word is required to convert the filled DOCX template to PDF. The program will not be able to generate scorecards without it.

Q: What file formats are supported?
A: Word templates must be in .docx format. Data files must be .csv. The optional back design must be a .pdf.

Q: Where are my generated PDFs saved?
A: All generated PDFs, as well as downloaded CSV templates and DOCX files, are saved to your Downloads folder.

Q: How do I format placeholders in my Word template?
A: Use clear, unique text (e.g. TEAMNAME, DATEHERE). For multi-card pages, the program automatically handles the _1, _2 suffixes — do not add them to the template placeholders.
